Overview
--------
3D force-density estimators and utilities for **RevelsMD**.

This module provides the :class:`~revelsMD.revels_3D.Revels3D` class, which
builds three-dimensional force and number density fields from MD trajectories
using the reduced-variance force-sampling framework.
